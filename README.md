# Awesome Newsletter [![Awesome](https://cdn.rawgit.com/sindresorhus/awesome/d7305f38d29fed78fa85652e3a63e154dd8e8829/media/badge.svg)](https://github.com/sindresorhus/awesome) [![Build Status](https://api.travis-ci.org/wallies/awesome-newsletters.svg)](https://travis-ci.org/wallies/awesome-newsletters)

A curated list of awesome newsletters.

Inspired by the `awesome-*` trend on GitHub.

The goal is to build a categorized community-driven collection of very well-known development newsletters.

Sharing, suggestions and contributions are always welcome! Please take a look at the [contribution guidelines](https://github.com/wallies/awesome-newsletters/blob/master/CONTRIBUTING.md) first.
